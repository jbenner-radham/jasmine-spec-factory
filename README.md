Jasmine Spec Generator
======================
An opinionated Jasmine spec generator for AMD modules.

Installation
------------
```sh
$ npm install && chmod +x cli.js
```

Usage
-----
```sh
./cli.js ~/Documents/path/to/an/amd/module.js
```

Testing
-------
```sh
$ npm test
```
